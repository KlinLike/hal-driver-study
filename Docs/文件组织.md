# 仓库文件组织说明

本文说明本工程顶层目录的职责划分，便于快速定位代码。目标芯片为 **STM32F103C8**，工具链以 **Keil MDK**（`MDK-ARM/`）为主。

## 顶层目录一览

| 目录 / 文件 | 说明 |
|-------------|------|
| `App/` | **应用层**：与板子强相关的业务逻辑、界面与协议（配置宏、OLED 时钟、按键计数、串口回显等）。 |
| `Core/` | **HAL / Cube 生成核心**：`main.c` 入口、中断向量实现、`gpio` / `i2c` / `usart` 初始化、`main.h` 板级引脚命名等。 |
| `Middlewares/` | **通用中间件**：与具体功能无关的可复用模块（环形队列、软定时器）。 |
| `Drivers/` | **厂商与板级驱动**：ST HAL 库、CMSIS、`MCU_F103` 下的 OLED 等外设封装。 |
| `MDK-ARM/` | **Keil 工程**：`.uvprojx`、启动文件、编译输出目录等。 |
| `Docs/` | **说明文档**：文件组织、功能说明等（与 `App/`、`Drivers/` 等目录命名风格统一）。 |
| `hal-driver-study.ioc` | **CubeMX 工程**（引脚与外设图形配置，重新生成时注意保留 `USER CODE` 区）。 |

## `App/`（应用层）

| 路径 | 作用 |
|------|------|
| `App/inc/app_config.h` | 运行模式（时钟 / 按键计数 / UART 中断演示）、OLED 布局、按键消抖时间等编译期配置。 |
| `App/inc/app.h` + `App/src/app.c` | 应用总入口：`app_init`、`app_poll`、软定时器分发、按键 EXTI 触发消抖定时器。 |
| `app_board` | LED、蜂鸣器、光敏等板级 GPIO 封装。 |
| `app_clock` | OLED 时钟显示、秒计时、1 Hz LED 翻转（仅时钟模式）。 |
| `app_key_count` | 按键计数模式下的计数与 OLED 更新。 |
| `app_ui` | 按当前模式整屏刷新 OLED。 |
| `app_uart_echo` | USART1 单字节中断接收 + 环形队列，主循环格式化回发。 |
| `app_uart_dma` | USART1 DMA+IDLE 不定长接收（双缓冲），主循环回显帧数据。 |

**原则**：业务扩展优先在 `App/` 增加模块，并在 `app.c` 中完成初始化与主循环调度；避免把应用协议塞进 `Middlewares/`。

## `Core/`

| 路径 | 作用 |
|------|------|
| `Core/Src/main.c` | 复位后初始化、调用 `app_init()`、主循环 `app_poll()`；按键 EXTI 回调中调用 `app_on_key_exti()`。 |
| `Core/Src/stm32f1xx_it.c` | 中断服务：`SysTick`（含 `app_systick_handler`）、`EXTI`、`USART1` 等。 |
| `Core/Src/gpio.c` 等 | 由 Cube 生成的 GPIO / I2C / USART 初始化代码。 |
| `Core/Inc/main.h` | 板级宏：`LED_GPIO_Port` / `KEY_GPIO_Pin` 等与 HAL 一致的命名。 |

## `Middlewares/`

| 文件 | 作用 |
|------|------|
| `circle_buffer.*` | 环形队列（单字节读写、容量查询）。 |
| `soft_timer.*` | 基于 SysTick 1 ms 轮询的软定时器（周期 / 单次）。 |

## `Drivers/`

| 路径 | 作用 |
|------|------|
| `Drivers/STM32F1xx_HAL_Driver/` | STM32F1 HAL 源码（由工程勾选模块决定参与编译）。 |
| `Drivers/CMSIS/` | CMSIS 核心与外设访问头文件等。 |
| `Drivers/MCU_F103/` | 本板 OLED 等外设驱动（如 `driver_oled.c`）。 |
